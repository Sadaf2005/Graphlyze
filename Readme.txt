Programming Language Used: HTML, CSS, JavaScript

Library Used: Cytoscape.js (version 3.10.2)

Cytoscape.min.js file is already present in the folder so no need of any software or header file. No internet connection is needed, only requirement is a browser 
(*** Preferably Google Chrome ***)


*** Instructions to MAKE A CUSTOM GRAPH and Visualise the algorithm ***

1. Click on Add Node option .
2.Then Click anywhere on the draw area to generate nodes.
3.Fill Weight,From,To node id value.
4. Click on Add Edge to generate edge.
5.Clear All to clean the draw area



*** Just run index.html file ***

*** Follow the instructions on the sidebar. ***
